# Omegon Generator
## Adaptable Wordlist Generator

### One-sentence description
Easy-to-use, high performance cross-platform application for creating permutations of user-defined sets of words for use in hash cracking.

### Additional information about the project
Omegon Generator requires that the user provide it a list of words, or use a premade one included with the program.
The user then uses the GUI of the application to drag "blocks" representing different addons.
These can be character sets, case variants, or even multiple wordlists. 
After the user runs the program, it outputs a file in a specified location with every possible variant of every possible word based on the rules provided.
